# Tree

`Tree` renders a hierarchical data structure with expand/collapse, keyboard navigation, and type-ahead search. Common use cases include file explorers, category trees, and nested menu structures. Selected items are indicated with a semibold label and accent background. Focused items show a ring indicator.

## Import

```tsx
import { Tree } from '@clidey/ux';
import type { TreeDataItem } from '@clidey/ux';
```

## Basic tree

```tsx
const treeData: TreeDataItem[] = [
  {
    id: '1',
    name: 'src',
    children: [
      {
        id: '1-1',
        name: 'components',
        children: [
          { id: '1-1-1', name: 'Button.tsx' },
          { id: '1-1-2', name: 'Card.tsx' },
          { id: '1-1-3', name: 'Input.tsx' },
        ],
      },
      {
        id: '1-2',
        name: 'lib',
        children: [
          { id: '1-2-1', name: 'utils.ts' },
          { id: '1-2-2', name: 'api.ts' },
        ],
      },
    ],
  },
  { id: '2', name: 'package.json' },
  { id: '3', name: 'tsconfig.json' },
];

<Tree data={treeData} />
```

## With selection

```tsx
function FileExplorer() {
  const [selected, setSelected] = useState<TreeDataItem | undefined>();

  return (
    <div className="flex gap-4">
      <div className="w-56 border rounded-lg p-2">
        <Tree
          data={treeData}
          onSelectChange={setSelected}
        />
      </div>
      <div className="flex-1 p-4">
        {selected ? (
          <p>Selected: <code>{selected.name}</code></p>
        ) : (
          <p className="text-muted-foreground">Select a file</p>
        )}
      </div>
    </div>
  );
}
```

## With default selection

```tsx
<Tree
  data={treeData}
  initialSelectedItemId="1-1-1"
  onSelectChange={(item) => console.log(item)}
/>
```

## Expand all by default

```tsx
<Tree data={treeData} expandAll />
```

## Custom icons

Override the default folder and item icons:

```tsx
import { Folder, FileText } from 'lucide-react';

<Tree
  data={treeData}
  folderIcon={Folder}
  itemIcon={FileText}
/>
```

## With per-item icons

Set an `icon` on individual `TreeDataItem` entries:

```tsx
import { FileJson, FileCode, FolderOpen } from 'lucide-react';

const treeData: TreeDataItem[] = [
  {
    id: '1',
    name: 'src',
    icon: FolderOpen,
    children: [
      { id: '1-1', name: 'index.ts', icon: FileCode },
      { id: '1-2', name: 'config.json', icon: FileJson },
    ],
  },
];
```

## Keyboard navigation

Click inside the tree to focus it, then use keyboard shortcuts:

| Key | Action |
| --- | --- |
| `‚Üë` / `‚Üì` | Move focus between visible nodes |
| `‚Üí` | Expand collapsed folder, or move to first child |
| `‚Üê` | Collapse expanded folder, or move to parent |
| `Enter` / `Space` | Select the focused node |
| `Home` / `End` | Jump to first / last visible node |
| Type a letter | Jump to next node whose name starts with that character |

Type-ahead resets after 500ms of inactivity, allowing multi-character search (e.g., type "re" to jump to "Reports").

## Controlled expand/collapse

Control which nodes are expanded externally (useful for search filtering):

```tsx
function SearchableTree() {
  const [expandedIds, setExpandedIds] = useState<string[]>([]);

  return (
    <Tree
      data={treeData}
      expandedIds={expandedIds}
      onExpandChange={setExpandedIds}
    />
  );
}
```

## Props

### Tree

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `data` | `TreeDataItem[] \| TreeDataItem` | ‚Äî | The tree data to render. |
| `initialSelectedItemId` | `string` | ‚Äî | ID of the item selected on initial render. |
| `onSelectChange` | `(item: TreeDataItem \| undefined) => void` | ‚Äî | Called when the selected item changes. |
| `expandAll` | `boolean` | `false` | Expands all nodes on initial render. |
| `expandedIds` | `string[]` | ‚Äî | Controlled expanded node IDs. |
| `onExpandChange` | `(ids: string[]) => void` | ‚Äî | Called when expand state changes. |
| `folderIcon` | `LucideIcon` | ‚Äî | Custom icon for folder nodes. |
| `itemIcon` | `LucideIcon` | ‚Äî | Custom icon for leaf nodes. |
| `className` | `string` | ‚Äî | Additional CSS classes. |

### TreeDataItem type

```tsx
type TreeDataItem = {
  id: string;
  name: string;
  icon?: LucideIcon;
  children?: TreeDataItem[];
};
```

<Note>
  Keyboard navigation follows the WAI-ARIA treeview pattern. Focus remains on the tree container; individual nodes are highlighted via `aria-activedescendant`.
</Note>
