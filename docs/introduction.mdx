# Clidey UX

**Clidey UX** is a production-ready React component library built on top of [Radix UI](https://radix-ui.com) primitives with [Tailwind CSS v4](https://tailwindcss.com) styling. It provides 37+ accessible, composable components designed for modern web applications.

## Why Clidey UX?

<CardGroup cols={3}>
  <Card title="Accessible by default" icon="shield-check" description="Every component is built on Radix UI primitives — keyboard navigable, screen reader compatible, and ARIA compliant out of the box." />
  <Card title="Fully typed" icon="code-2" description="Complete TypeScript definitions with strict mode. Your editor autocompletes props, variants, and event handlers." />
  <Card title="Dark mode built in" icon="moon" description="Every component supports light and dark mode through the ThemeProvider. No extra configuration needed." />
  <Card title="Composable API" icon="layers" description="Components use a sub-component composition pattern. Mix and match parts to build exactly the UI you need." />
  <Card title="Tailwind CSS v4" icon="paint-bucket" description="Styled with the latest Tailwind CSS v4. Customize with className or extend through your own Tailwind config." />
  <Card title="Zero runtime overhead" icon="zap" description="Pure React components with no runtime theme engine. Styles are compiled at build time for maximum performance." />
</CardGroup>

## What's included

The library ships **37 UI components** plus a complete theming system:

<AccordionGroup>
  <Accordion title="Actions" icon="pointer">
    `Button`, `ButtonGroup` — Primary interaction elements with multiple variants and sizes.
  </Accordion>
  <Accordion title="Inputs" icon="keyboard">
    `Input`, `TextArea`, `SearchInput`, `Checkbox`, `Switch`, `Select`, `SearchSelect`, `Label` — Form controls for all data entry needs.
  </Accordion>
  <Accordion title="Display" icon="layout">
    `Badge`, `Card`, `Alert`, `Spinner`, `Skeleton`, `Progress`, `EmptyState`, `Separator` — Components for presenting information and loading states.
  </Accordion>
  <Accordion title="Navigation" icon="map">
    `Tabs`, `Breadcrumb`, `Pagination`, `Sidebar` — Navigation patterns for multi-page and multi-section UIs.
  </Accordion>
  <Accordion title="Overlays" icon="layers">
    `Dialog`, `AlertDialog`, `Drawer`, `Sheet`, `Tooltip`, `Popover` — Floating and modal UI elements.
  </Accordion>
  <Accordion title="Menus" icon="menu">
    `DropdownMenu`, `ContextMenu`, `Command` — Contextual action menus and command palettes.
  </Accordion>
  <Accordion title="Layout" icon="columns">
    `Accordion`, `ResizablePanelGroup`, `ScrollArea`, `StackList` — Structural layout primitives.
  </Accordion>
  <Accordion title="Data & Visualization" icon="bar-chart-2">
    `Table` (with virtualization), `Chart` (Recharts wrapper), `Tree` — Components for displaying and exploring data.
  </Accordion>
  <Accordion title="Utilities" icon="settings">
    `Icon`, `Toaster` (Sonner) — Supporting utilities for icons and notifications.
  </Accordion>
</AccordionGroup>
